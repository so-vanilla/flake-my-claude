# SMART goal template

## Proposed SMART goal

<1〜3文で、達成状態と検証方法を書く。>

| SMART | 内容 |
|---|---|
| Specific | 何を変えるか |
| Measurable | 何で完了判定するか |
| Achievable | 現実的な範囲か |
| Relevant | ユーザー要求との対応 |
| Time bounded | 反復上限や停止条件 |

## Acceptance criteria

- [ ] 検証コマンドが成功する。
- [ ] 対象テストが通過する。
- [ ] 対象ファイルに意図した変更がある。
- [ ] 非対象ファイルが変更されていない。
- [ ] 最終報告に evidence がある。

## Stop conditions

- 成功条件をすべて満たしたら停止する。
- 同じ失敗が2回続いたら原因仮説を更新する。
- 上限に達したら、未解決事項を明示して停止する。
