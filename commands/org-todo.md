# /org-todo - チャットコンテキストからタスク追加

会話の文脈を分析してタスク候補を抽出し、ユーザー確認の上で `~/org/todo.org` に追記する。

## 引数

$ARGUMENTS: タスクのヒント（省略可。省略時は会話全体の文脈から抽出）

## 処理フロー

以下のステップを順に実行すること。

### 1. タスク候補の抽出

会話の文脈（および $ARGUMENTS があればそれ）を分析し、タスク候補をリスト化する:

- 各候補にはタイトルと DEADLINE の提案を含める
- DEADLINE の提案は会話から推測できる場合のみ付与（推測できなければ「なし」）
- 具体的なアクションとして成立するものだけを候補にする

### 2. ユーザーへの確認

各タスク候補について、AskUserQuestion ツールで以下の選択肢を提示:

1. **OK** - 提案通りに追加
2. **DEADLINEなし** - タイトルのみで追加（DEADLINE行を省略）
3. **タスク化しない** - この候補をスキップ
4. **カスタム** - タイトルや DEADLINE を自由に指定

### 3. todo.org への追記

- Readツールで `~/org/todo.org` の現在の内容を確認
- 確定したタスクをファイル末尾に追記する
- フォーマット:

```org
* TODO <タイトル>
DEADLINE: <YYYY-MM-DD DayOfWeek>
```

- DEADLINEなしの場合は DEADLINE行を省略:

```org
* TODO <タイトル>
```

### 4. 結果報告

以下の情報を報告:

- 追加したタスクの一覧（タイトルと DEADLINE）
- スキップされた候補があればその旨も報告
