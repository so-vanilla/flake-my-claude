# /org-agenda - 今日のアジェンダ表示

`~/org/schedule.org` と `~/org/todo.org` を読み込み、現在日時を基準に今日のアジェンダを合成・提示する。

**重要: このコマンドは読み取り専用。ファイルの作成・変更・削除は一切行わない。**

## 引数

なし

## 処理フロー

以下のステップを順に実行し、結果をまとめて報告する。

### 1. orgファイルの読み込み

- Readツールで `~/org/schedule.org` と `~/org/todo.org` を読み込む
- どちらかが存在しない場合はその旨を報告し、存在するファイルのみで続行

### 2. schedule.org の解析

- 今日の日付に該当するイベント・予定を抽出
- 時刻付きイベントは時系列順にソート

### 3. todo.org の解析

各エントリを以下のカテゴリに分類:

- **期限切れタスク**: DEADLINE が過去日の `TODO` タスク
- **本日期限タスク**: DEADLINE が今日の `TODO` タスク
- **近日タスク**: DEADLINE が明日〜7日以内の `TODO` タスク
- **繰り返しタスク**: `++1d` を含む本日分のデイリータスク

### 4. アジェンダの合成・提示

以下の構造で報告する:

```
## 📅 今日のアジェンダ（YYYY-MM-DD DayOfWeek）

### スケジュール
<schedule.org から抽出した今日のイベント（時刻順）>

### 期限切れ（要対応）
<DEADLINE が過去日の TODO タスク。日数も表示>

### 本日期限
<今日が DEADLINE の TODO タスク>

### デイリータスク
<++1d の繰り返しタスク>

### 今週の予定
<明日〜7日以内に DEADLINE がある TODO タスク>

### サマリー
<全体の状況を簡潔に要約。優先度が高いものへのアドバイス>
```

カテゴリにエントリがない場合はそのセクションを省略する。
