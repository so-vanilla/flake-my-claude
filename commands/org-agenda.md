# /org-agenda - 今日のアジェンダ表示

`~/org/schedule.org` と `~/org/todo.org` を読み込み、現在日時を基準に今日のアジェンダを合成・提示する。

**重要: このコマンドは読み取り専用。ファイルの作成・変更・削除は一切行わない。**

## 引数

なし

## 実行方法

Task ツールで以下の通り subagent を起動し、全処理を委譲せよ:
- subagent_type: `"general-purpose"`
- model: `"haiku"`
- prompt: 以下のプロンプト全文。`{TODAY}` は実際の今日の日付（YYYY-MM-DD 曜日）に置換すること

subagent の出力をそのままユーザーに表示すること。

---

### subagent prompt

```
あなたは org-mode ファイルを解析してアジェンダを生成するエージェントです。
今日の日付は {TODAY} です。

以下の手順を順に実行してください。

## 1. orgファイルの読み込み

Read ツールで以下の2ファイルを読み込む（並列可）:
- ~/org/schedule.org
- ~/org/todo.org

どちらかが存在しない場合はその旨を報告し、存在するファイルのみで続行する。

## 2. schedule.org の解析

- 今日の日付に該当するイベント・予定を抽出する
- 時刻付きイベントは時系列順にソートする

## 3. todo.org の解析

各エントリを以下のカテゴリに分類する:

- **期限切れタスク**: DEADLINE が過去日の TODO タスク
- **本日期限タスク**: DEADLINE が今日の TODO タスク
- **近日タスク**: DEADLINE が明日〜7日以内の TODO タスク
- **繰り返しタスク**: ++1d を含む本日分のデイリータスク

## 4. アジェンダの合成・提示

以下の構造で出力する:

## 今日のアジェンダ（{TODAY}）

### スケジュール
<schedule.org から抽出した今日のイベント（時刻順）>

### 期限切れ（要対応）
<DEADLINE が過去日の TODO タスク。日数も表示>

### 本日期限
<今日が DEADLINE の TODO タスク>

### デイリータスク
<++1d の繰り返しタスク>

### 今週の予定
<明日〜7日以内に DEADLINE がある TODO タスク>

### サマリー
<全体の状況を簡潔に要約。優先度が高いものへのアドバイス>

カテゴリにエントリがない場合はそのセクションを省略する。
日本語で出力すること。
```
